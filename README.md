# Rawkuma scraper 
A simple Python script that downloads manga from Rawkuma by using wget via subprocess to fetch each chapter's zip archive from Rawkuma's Google Drive.

## Requirements
- Any Linux distro
- Python 3.x
- BeautifulSoup4
- Requests
